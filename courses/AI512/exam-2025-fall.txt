## Question 1: Bayesian Cancer Screening  (5 questions)
--------------------------------------------------

Setup: $P(\text{cancer}) = 0.01$, $P(+|\text{cancer}) = 0.90$, $P(+|\neg\text{cancer}) = 0.03$.
So $P(+) = 0.90 \cdot 0.01 + 0.03 \cdot 0.99 = 0.009 + 0.0297 = 0.0387$ and
$P(\text{cancer}|+) = 0.009 / 0.0387 \approx 0.233$.

1. $P(\text{cancer} | +) > $ false positive rate of the test. Is this TRUE or FALSE?
   1) [CORRECT] True
        -> $P(\text{cancer}|+) \approx 0.233 \gg 0.03$ (the false positive rate).
   2) [WRONG]   False
        -> Bayes' theorem pushes the posterior well above the raw false positive rate.

2. $P(\text{positive test}) < P(\text{cancer})$. Is this TRUE or FALSE?
   1) [CORRECT] False
        -> $P(+) = 0.0387 > P(\text{cancer}) = 0.01$; false positives inflate the positive rate.
   2) [WRONG]   True
        -> The positive rate includes false positives, making it larger than the cancer prevalence.

3. $P(\neg\text{cancer} | -) < P(\neg\text{cancer})$. Is this TRUE or FALSE?
   1) [CORRECT] False
        -> $P(\neg\text{cancer}|-) \approx 0.999 > P(\neg\text{cancer}) = 0.99$; a negative test makes cancer even less likely.
   2) [WRONG]   True
        -> A negative result raises (not lowers) the probability of not having cancer.

4. $P(\text{positive test}) < P(\text{cancer}|+)$. Is this TRUE or FALSE?
   1) [CORRECT] True
        -> $0.0387 < 0.233$; the marginal positive rate is much smaller than the posterior cancer probability.
   2) [WRONG]   False
        -> The posterior $P(\text{cancer}|+) \approx 0.233$ is indeed larger than $P(+) = 0.0387$.

5. $P(\text{cancer}, +) < P(\text{cancer}|+)$. Is this TRUE or FALSE?
   1) [CORRECT] True
        -> $P(\text{cancer}, +) = 0.009 < P(\text{cancer}|+) \approx 0.233$; a joint probability is always $\leq$ a conditional probability here.
   2) [WRONG]   False
        -> The joint probability $P(A,B) \leq P(A|B)$ whenever $P(B) \leq 1$.


## Question 2: Expectation and Variance  (5 questions)
--------------------------------------------------

1. For all joint distributions: $\mathbb{E}[X+Y] = \mathbb{E}[X] + \mathbb{E}[Y]$. TRUE or FALSE?
   1) [CORRECT] True
        -> Linearity of expectation holds unconditionally, regardless of dependence.
   2) [WRONG]   False
        -> This is one of the most fundamental properties of expectation and requires no independence.

2. For all joint distributions: $\text{Var}[X+Y] = \text{Var}[X] + \text{Var}[Y]$. TRUE or FALSE?
   1) [CORRECT] False
        -> $\text{Var}[X+Y] = \text{Var}[X] + \text{Var}[Y] + 2\,\text{Cov}[X,Y]$; equality holds only when $X \perp Y$.
   2) [WRONG]   True
        -> The covariance term vanishes only under independence.

3. For all joint distributions: $\mathbb{E}[X] = \mathbb{E}_Y[\mathbb{E}_X[X|Y]]$. TRUE or FALSE?
   1) [CORRECT] True
        -> This is the law of total expectation (tower property).
   2) [WRONG]   False
        -> The tower property holds for all joint distributions without any independence assumption.

4. For all joint distributions: $\text{Var}[X] = \mathbb{E}_Y[\text{Var}_X[X|Y]] + \text{Var}_Y[\mathbb{E}_X[X|Y]]$. TRUE or FALSE?
   1) [CORRECT] True
        -> This is the law of total variance (Eve's law); it holds universally.
   2) [WRONG]   False
        -> Both terms are non-negative and together they always sum to the marginal variance.

5. For all joint distributions: $\mathbb{E}[X \times Y] = \mathbb{E}[X] \times \mathbb{E}[Y]$. TRUE or FALSE?
   1) [CORRECT] False
        -> This holds only when $X$ and $Y$ are uncorrelated; dependent variables can have $\mathbb{E}[XY] \neq \mathbb{E}[X]\mathbb{E}[Y]$.
   2) [WRONG]   True
        -> A counterexample: $X = Y \sim \text{Bernoulli}(0.5)$; then $\mathbb{E}[XY] = 0.5 \neq 0.25 = \mathbb{E}[X]\mathbb{E}[Y]$.

## Question 4: Maximum Likelihood Estimation  (5 questions)
--------------------------------------------------

1. The MLE for $\mu$ is the sample mean $\frac{1}{n}\sum_{i=1}^n x_i$. TRUE or FALSE?
   1) [CORRECT] True
        -> Maximising the Gaussian log-likelihood over $\mu$ yields the sample mean.
   2) [WRONG]   False
        -> Setting $\partial \ell / \partial \mu = 0$ directly gives $\hat{\mu} = \bar{x}$.

2. The MLE for variance is an unbiased estimator of the true population variance. TRUE or FALSE?
   1) [CORRECT] False
        -> The MLE is $\hat{\sigma}^2 = \frac{1}{n}\sum(x_i-\hat{\mu})^2$, which is biased by a factor $(n-1)/n$.
   2) [WRONG]   True
        -> The unbiased estimator uses $n-1$ in the denominator (Bessel's correction), not $n$.

3. If the true mean $\mu$ were known, the MLE for variance would be $\frac{1}{n-1}\sum_{i=1}^n (x_i-\mu)^2$. TRUE or FALSE?
   1) [CORRECT] False
        -> With $\mu$ known, the MLE is $\frac{1}{n}\sum(x_i-\mu)^2$ (no Bessel's correction needed).
   2) [WRONG]   True
        -> The $1/(n-1)$ correction appears only when $\mu$ is estimated; here it is known, so $1/n$ is correct.

4. The log-likelihood function is a linear function of $\sigma^2$. TRUE or FALSE?
   1) [CORRECT] False
        -> $\ell \propto -\frac{n}{2}\ln\sigma^2 - \frac{1}{2\sigma^2}\sum(x_i-\mu)^2$; both $\ln\sigma^2$ and $1/\sigma^2$ are nonlinear.
   2) [WRONG]   True
        -> The log-likelihood contains $\ln\sigma^2$ and $1/\sigma^2$ terms, which are not linear.

5. The maximum of the likelihood $L$ is achieved at the same $(\mu, \sigma^2)$ as the maximum of $\log L$. TRUE or FALSE?
   1) [CORRECT] True
        -> $\log$ is a strictly increasing function, so $\arg\max L = \arg\max \log L$.
   2) [WRONG]   False
        -> Monotone transformations preserve the location of the maximum.

## Question 7: Bias–Variance Trade-off  (5 questions)
--------------------------------------------------

1. As max_depth $\to \infty$, training error decreases but variance typically increases. TRUE or FALSE?
   1) [CORRECT] True
        -> A deeper tree memorises noise, lowering training error while increasing sensitivity to the specific dataset.
   2) [WRONG]   False
        -> This is the canonical overfitting behaviour: training error falls, generalisation variance rises.

2. A decision tree with max_depth = 1 has higher variance and lower bias than deeper trees. TRUE or FALSE?
   1) [CORRECT] False
        -> A depth-1 stump is a highly constrained model: high bias, low variance.
   2) [WRONG]   True
        -> Simpler models (smaller depth) have more bias and less variance, not the reverse.

3. Increasing the minimum number of samples required to split a node increases model bias. TRUE or FALSE?
   1) [CORRECT] True
        -> A higher minimum forces coarser splits, reducing model flexibility and increasing systematic underfitting (bias).
   2) [WRONG]   False
        -> Any regularisation that constrains the tree reduces variance but increases bias.

4. The bias component of error represents the sensitivity of predictions to small fluctuations in the training set. TRUE or FALSE?
   1) [CORRECT] False
        -> That is the definition of variance. Bias is the systematic error from incorrect assumptions about the function being learned.
   2) [WRONG]   True
        -> Sensitivity to training-set fluctuations is variance, not bias.

5. If a decision tree perfectly classifies every noisy training example, the test error is dominated by variance and irreducible error. TRUE or FALSE?
   1) [CORRECT] True
        -> Perfect training accuracy on noisy data signals overfitting; the dominant test-error terms are variance (fitting noise) and irreducible (label noise).
   2) [WRONG]   False
        -> Perfect training accuracy on noisy data is exactly the scenario where variance and irreducible error dominate.

## Question 10: Empirical Risk Minimisation  (5 questions)
--------------------------------------------------

1. $R_{\text{emp}}(h)$ is an unbiased estimator of $R(h)$ for a fixed $h$, but is biased for the ERM hypothesis $\hat{h}$. TRUE or FALSE?
   1) [CORRECT] True
        -> For a fixed $h$, $\mathbb{E}[R_{\text{emp}}(h)] = R(h)$. But $\hat{h}$ is selected to minimise $R_{\text{emp}}$, causing optimistic bias.
   2) [WRONG]   False
        -> Selection bias: choosing $\hat{h}$ to minimise training error makes $R_{\text{emp}}(\hat{h})$ a downward-biased estimate of $R(\hat{h})$.

2. The generalisation gap is guaranteed to decrease at a linear rate as $n$ increases. TRUE or FALSE?
   1) [CORRECT] False
        -> Finite-class bounds give a gap of $O(1/\sqrt{n})$ (sub-linear), not $O(1/n)$ (linear).
   2) [WRONG]   True
        -> Standard PAC bounds decay as $\sqrt{1/n}$, not $1/n$.

3. If $\mathcal{H} \subseteq \mathcal{H}'$, then $R_{\text{emp}}(\hat{h}') \leq R_{\text{emp}}(\hat{h})$. TRUE or FALSE?
   1) [CORRECT] True
        -> $\mathcal{H}'$ contains all of $\mathcal{H}$, so the ERM on $\mathcal{H}'$ can always do at least as well on training data.
   2) [WRONG]   False
        -> ERM on a strictly larger class can only keep or lower the empirical error; it cannot increase it.

4. To ensure $R(\hat{h}) \leq R_{\text{emp}}(\hat{h}) + \epsilon$ with confidence $1-\delta$, $n$ must scale linearly with $|\mathcal{H}|$. TRUE or FALSE?
   1) [CORRECT] False
        -> The required $n$ scales as $\ln|\mathcal{H}|$ (logarithmically), not linearly.
   2) [WRONG]   True
        -> From the finite-class bound: $n \geq \frac{1}{2\epsilon^2}(\ln|\mathcal{H}|+\ln(1/\delta))$, which is logarithmic in $|\mathcal{H}|$.

5. The bound $R(\hat{h}) \leq R_{\text{emp}}(\hat{h}) + \sqrt{\frac{\ln|\mathcal{H}|+\ln(1/\delta)}{2n}}$ holds for all finite $\mathcal{H}$ but not necessarily all distributions. TRUE or FALSE?
   1) [CORRECT] False
        -> The bound is distribution-free; it holds for all finite $\mathcal{H}$ and for all data distributions.
   2) [WRONG]   True
        -> The derivation (union bound + Hoeffding) places no restriction on the distribution.


## Question 11: No-Free-Lunch Theorem  (5 questions)
--------------------------------------------------

1. If algorithm A has smaller generalisation error than B on one dataset, A is expected to outperform B across all distributions. TRUE or FALSE?
   1) [CORRECT] False
        -> NFL says no algorithm is universally superior; for every distribution where A beats B, there is another where B beats A.
   2) [WRONG]   True
        -> This directly contradicts the No-Free-Lunch theorem.

2. For any learning algorithm, there exists at least one distribution where its error cannot be reduced by enlarging the training set. TRUE or FALSE?
   1) [CORRECT] True
        -> NFL guarantees the existence of an adversarial distribution where any algorithm performs no better than random, regardless of sample size.
   2) [WRONG]   False
        -> This is precisely what the NFL theorem establishes.

3. NFL implies that inductive bias is necessary for better-than-random generalisation on any specific task. TRUE or FALSE?
   1) [CORRECT] True
        -> Without assumptions (inductive bias) about the target distribution, no algorithm can generalise better than chance in the worst case.
   2) [WRONG]   False
        -> The NFL theorem is the formal argument that inductive bias is unavoidable for learning.

4. The NFL theorem applies only to parametric predictors, not non-parametric models like k-NN. TRUE or FALSE?
   1) [CORRECT] False
        -> NFL applies to all learning algorithms regardless of whether they are parametric or non-parametric.
   2) [WRONG]   True
        -> The theorem makes no restriction on the class of algorithms; k-NN is covered too.

5. NFL implies that for any distribution with a perfect classifier, any algorithm converges to zero generalisation error if $m \geq |\mathcal{X}|/8$. TRUE or FALSE?
   1) [CORRECT] False
        -> NFL says the opposite: even when a perfect classifier exists, there are distributions where any algorithm can fail. NFL gives no convergence guarantee.
   2) [WRONG]   True
        -> NFL is an impossibility result, not a convergence guarantee.

## Question 13: VC Dimension of Axis-Aligned Boxes in $\mathbb{R}^3$  (5 questions)
--------------------------------------------------

1. There exists a set of 6 points in $\mathbb{R}^3$ that can be shattered by $\mathcal{H}$. TRUE or FALSE?
   1) [CORRECT] True

   2) [WRONG]   False

2. $\text{VC}(\mathcal{H}) = 4$ in $\mathbb{R}^3$. TRUE or FALSE?
   1) [CORRECT] False
        -> Each axis contributes 2 degrees of freedom (min and max); so $\text{VC} = 2 \times 3 = 6$, not 4.
   2) [WRONG]   True
        -> The formula is $2d$, not $d+1$; in 3D that gives 6.

3. There exists a set of 4 points in $\mathbb{R}^3$ for which every labelling is achievable by some box. TRUE or FALSE?
   1) [CORRECT] True
        -> Since $\text{VC}(\mathcal{H}) = 6 \geq 4$, any set of 4 points from the shattered 6-point set can be shattered.
   2) [WRONG]   False
        -> A VC dimension of 6 guarantees that sets of size $\leq 6$ can be shattered (for the right configuration).

4. For $n > 10$, the growth function satisfies $\Pi_\mathcal{H}(n) < 2^n$. TRUE or FALSE?
   1) [CORRECT] True
        -> By Sauer's lemma, for $n > \text{VC}(\mathcal{H}) = 6$, the growth function is strictly less than $2^n$. $10 > 6$ suffices.
   2) [WRONG]   False
        -> Sauer's lemma guarantees $\Pi_\mathcal{H}(n) < 2^n$ for all $n > \text{VC}(\mathcal{H})$, so certainly for $n > 10$.

5. There does not exist a set of 5 points in $\mathbb{R}^3$ that can be shattered by $\mathcal{H}$. TRUE or FALSE?
   1) [CORRECT] False
        -> $\text{VC}(\mathcal{H}) = 6$ implies there are shatterable sets of size 5 (any 5 from the 6-point shattered set).
   2) [WRONG]   True
        -> The VC dimension of 6 means sets of size 5 can certainly be shattered.
